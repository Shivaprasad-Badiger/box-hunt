# How to start and run

1. Change into the app folder
   ```
   cd box-hunt
   ```
   - Moves your terminal into the project directory that contains package.json.

2. Install dependencies
   ```
   npm install
   ```
   - Downloads and installs all packages listed in package.json to node_modules.

3. Start the development server
   ```
   npm run dev
   ```
   - Launches Vite's dev server (HMR). Open the URL shown in the terminal (default: http://localhost:5173).
